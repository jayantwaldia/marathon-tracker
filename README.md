# Ladakh 3:45 — Marathon Tracker

A 26-week marathon training plan tracker for the Ladakh Marathon (Sep 13, 2026). Track your workouts, log distance and time, and monitor your progress toward a 3:45 finish.

## Features

- **26-week training plan** — Base building, aerobic development, race-specific prep, and taper
- **Workout logging** — Log distance, time, mood, and notes for each run
- **Progress tracking** — Weekly completion, total km, and phase progress
- **Race day tips** — Target pace, acclimatization, and nutrition guidance
- **Mobile-friendly** — Works great on phones; progress saved in browser

## Live Site

Hosted on GitHub Pages: **https://jayantwaldia.github.io/marathon-tracker/**

## Local Development

Open `index.html` in a browser, or run a local server:

```bash
# Using Python
python3 -m http.server 8000

# Using Node.js (npx)
npx serve .
```

Then visit http://localhost:8000
